# Project LaTeX website thể dục - chuẩn định dạng

Mở `main.tex` bằng TeXstudio và biên dịch bằng XeLaTeX.

Các điểm đã chuẩn hóa:
- Mục lục tự động (`\tableofcontents`) có dấu chấm dẫn dòng.
- Số trang đặt ở chân trang giữa.
- Danh mục hình và danh mục bảng tự động.
- Bảng và hình đánh số theo chương/phụ lục.
- Không dùng `\setcounter{table}{...}` để ép số bảng.
- Bìa ngoài, bìa trong và lời cam đoan căn theo template tham chiếu.
